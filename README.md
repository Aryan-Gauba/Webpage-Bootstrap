# Move It! - Professional Moving Services

A modern, responsive landing page built for a moving services company. This project was developed as part of The Complete 2024 Web Development Bootcamp by Dr. Angela Yu, focusing on mastering the **Bootstrap 5** framework for rapid UI development.


## 🚀 Features

- **Responsive Design:** Fully optimized for desktops, tablets, and mobile devices using the Bootstrap grid system.
- **Modern UI/UX:** Clean aesthetic with a focus on call-to-action (CTA) buttons and service highlights.
- **Interactive Elements:** Features a responsive navigation bar, hover effects, and themed components.
- **Optimized Layout:** Uses Bootstrap's Flexbox utilities and pre-built components like Cards, Buttons, and Navs.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure.
- **CSS3:** Custom styling and overrides.
- **Bootstrap 5:** Frontend framework for layout and responsiveness.
- **Google Fonts:** Integration for professional typography.
